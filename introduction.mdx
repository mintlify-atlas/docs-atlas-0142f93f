The Okta Go Management SDK (`okta-sdk-golang`) gives you full programmatic access to the Okta Management API from Go server-side applications. Whether you're automating user provisioning, managing application assignments, or building complex policy workflows, this SDK handles authentication, pagination, rate limiting, and caching so you can focus on your business logic.

## SDK highlights

- **Multiple auth modes** — API Token (SSWS), OAuth 2.0 with Private Key, JWT assertion, or Bearer Token
- **Built-in caching** — memory cache with configurable TTL reduces API calls automatically
- **Rate limit handling** — automatic retry with backoff on 429 responses
- **Pagination** — simple `HasNextPage()` / `Next()` helpers for all list endpoints
- **DPoP support** — Demonstrating Proof of Possession for enhanced OAuth 2.0 security
- **Context-aware** — pass `context.Context` to every API call for cancellation and deadlines

## Current version

The current stable release is **v6.x**. See the [migration guide](/migration/v5-to-v6) if you're upgrading from an earlier version.

<Note>
  This SDK is for the **Okta Management API** only. To call the Authentication API or implement end-user sign-in flows, use the [Okta JWT Verifier for Go](https://github.com/okta/okta-jwt-verifier-golang) or implement the OAuth 2.0 flows directly.
</Note>
