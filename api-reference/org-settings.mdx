The org settings APIs let you read and update configuration for your Okta organization. The Go SDK exposes these capabilities through four focused interfaces: `OrgSettingGeneralAPI` for core org data, `OrgSettingAdminAPI` for admin console behavior, `OrgSettingCommunicationAPI` for email opt-in preferences, and `OrgSettingCustomizationAPI` for end-user UI preferences. Each interface is accessed via the corresponding property on the `*okta.APIClient`.

## OrgSetting model

`OrgSetting` represents the general configuration of an Okta organization.

</ResponseField>

<ResponseField name="subdomain" type="string">

</ResponseField>

<ResponseField name="website" type="string">

</ResponseField>

## OrgSettingGeneralAPI

Accessed via `client.OrgSettingGeneralAPI()`. Provides read and write access to the core org settings at `GET/POST/PUT /api/v1/org`.

### GetOrgSettings

Retrieves the general settings for your Okta org.

```go
func (a *OrgSettingGeneralAPIService) GetOrgSettings(ctx context.Context) ApiGetOrgSettingsRequest
```

<ParamField path="ctx" type="context.Context" required>
  Request context for authentication, cancellation, and deadlines.
</ParamField>

**Returns** `(*OrgSetting, *APIResponse, error)`

---

### UpdateOrgSettings

Applies a partial update to the org general settings. Only the fields you set on the request body are changed.

```go
func (a *OrgSettingGeneralAPIService) UpdateOrgSettings(ctx context.Context) ApiUpdateOrgSettingsRequest
```

<ParamField path="ctx" type="context.Context" required>
  Request context.
</ParamField>

Chain `.OrgSetting(okta.OrgSetting{...})` on the returned request object to supply the body, then call `.Execute()`.

**Returns** `(*OrgSetting, *APIResponse, error)`

---

### ReplaceOrgSettings

Replaces all org general settings with the values you provide.

```go
func (a *OrgSettingGeneralAPIService) ReplaceOrgSettings(ctx context.Context) ApiReplaceOrgSettingsRequest
```

<ParamField path="ctx" type="context.Context" required>
  Request context.
</ParamField>

Chain `.OrgSetting(okta.OrgSetting{...})` on the returned request object to supply the full replacement body, then call `.Execute()`.

**Returns** `(*OrgSetting, *APIResponse, error)`

## Example: reading and updating org settings

The following example retrieves the current org settings, updates the company website URL, and writes the change back using `UpdateOrgSettings`.

```go
package main

import (
	"context"
	"fmt"
	"log"

	okta "github.com/okta/okta-sdk-golang/v6/okta"
)

func main() {
	cfg, err := okta.NewConfiguration()
	if err != nil {
		log.Fatal(err)
	}
	client := okta.NewAPIClient(cfg)
	ctx := context.Background()

	// Retrieve current org settings
	settings, _, err := client.OrgSettingGeneralAPI().
		GetOrgSettings(ctx).
		Execute()
	if err != nil {
		log.Fatal(err)
	}
	fmt.Printf("Org: %s (subdomain: %s)\n", *settings.CompanyName, *settings.Subdomain)

	// Update the website URL
	newWebsite := "https://example.com"
	updated := okta.OrgSetting{}
	updated.SetWebsite(newWebsite)

	result, _, err := client.OrgSettingGeneralAPI().
		UpdateOrgSettings(ctx).
		OrgSetting(updated).
		Execute()
	if err != nil {
		log.Fatal(err)
	}
	fmt.Printf("Updated website: %s\n", *result.Website)
}
```
